# 📲 Install App in 3 Simple Steps

---

## 🎯 STEP 1: Open Profile

**Tap the Profile icon at the bottom right corner**

```
Bottom of screen:
[Home] [Pay] [Rewards] [History] [Profile] ← TAP THIS
```

---

## 🎯 STEP 2: Tap "Install App"

**Scroll down and find "Install App" option**

```
Look for:
📥 Install App
   Add this app to your device...

↑ TAP THIS
```

---

## 🎯 STEP 3: Tap "Install"

**In the dialog that appears, tap the purple "Install" button**

```
Dialog shows:
📥 Install Payment App

[Cancel]  [Install] ← TAP THIS
```

---

## ✅ DONE!

### **What happens next:**

**Chrome/Edge (Android):**
- Browser prompt appears
- Tap "Add" or "Install"
- ✅ App icon on home screen!

**Safari (iPhone):**
- Instructions appear
- Follow the 4 steps shown
- ✅ App icon on home screen!

---

## 🚀 Open Your App

**Find "GPay Demo" icon on your home screen and tap it!**

✅ Opens in full-screen  
✅ Works offline  
✅ Looks like a real app  

---

## 🎉 That's It!

**Just 3 taps:**
1. Profile (bottom right)
2. Install App (in settings)
3. Install (purple button)

**Enjoy your payment app!** 🚀
