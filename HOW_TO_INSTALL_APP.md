# 📲 How to Install the App on Mobile

## 🎯 Step-by-Step Installation Guide

---

## 📱 Method 1: Install from Profile Settings

### **Step 1: Open Profile**
```
1. Open the app in your mobile browser
2. Tap the "Profile" icon at the bottom right
   (It's the last icon in the bottom navigation bar)
```

### **Step 2: Find Install Option**
```
3. Scroll down to the "Preferences" section
4. Look for "Install App" option
   (It has a purple download icon 📥)
5. Tap on "Install App"
```

### **Step 3: Install the App**

#### **On Chrome/Edge (Android):**
```
6. A dialog will appear with "Install Payment App"
7. Tap the "Install" button (purple button)
8. Native browser prompt will appear
9. Tap "Install" or "Add" in the browser prompt
10. ✅ App icon appears on your home screen!
```

#### **On Safari (iPhone):**
```
6. A dialog will appear with "Install Payment App"
7. Tap the "Install" button
8. Manual instructions will appear
9. Follow these steps:
   - Tap the Share button (↗️) at the bottom
   - Scroll down and tap "Add to Home Screen"
   - Tap "Add" in the top right
10. ✅ App icon appears on your home screen!
```

### **Step 4: Open the Installed App**
```
11. Go to your home screen
12. Find the "GPay Demo" app icon
13. Tap to open
14. ✅ App opens in full-screen (no browser UI)
15. ✅ Works offline!
```

## 🎉 What Happens After Installation

### **Automatic Install (Chrome/Edge):**
```
1. Native prompt appears
2. You tap "Install"
3. ✅ Success sound plays
4. ✅ Phone vibrates
5. ✅ Message: "App installed successfully! 🎉"
6. ✅ App icon on home screen
```

### **Manual Install (Safari):**
```
1. Instructions dialog appears
2. You follow the steps
3. ✅ App icon on home screen
```

### **Opening the Installed App:**
```
1. Tap app icon on home screen
2. ✅ Opens in full-screen
3. ✅ No browser address bar
4. ✅ Looks like native app
5. ✅ Works offline
```

## 🔍 How to Verify It's Installed

### **Check 1: Home Screen**
- Look for "GPay Demo" icon on your home screen
- Icon should be visible among other apps

### **Check 2: Full-Screen Mode**
- Open the app from home screen
- No browser address bar should be visible
- No browser controls (back, forward, refresh)
- Status bar shows app name

### **Check 3: Offline Test**
- Open the installed app
- Turn off WiFi and mobile data
- App should still work
- All screens should load

### **Check 4: Profile Settings**
- Open app and go to Profile
- Tap "Install App" option
- Should show "App Installed ✓" dialog
- Not asking to install again

## 🐛 Troubleshooting

### **Problem: Can't find "Install App" option**
**Solution:**
- Make sure you're on the Profile tab (bottom right)
- Scroll down to "Preferences" section
- Look for purple download icon

### **Problem: Native prompt doesn't appear**
**Solution:**
- Your browser might not support auto-install
- Tap "Show manual instructions" link
- Follow the step-by-step guide

### **Problem: App not on home screen**
**Solution:**
- Check if installation completed
- Look in app drawer (Android)
- Search for "GPay Demo" on home screen
- Try installing again

### **Problem: Opens in browser, not full-screen**
**Solution:**
- Make sure you're opening from home screen icon
- Not from browser bookmark
- Try reinstalling the app
